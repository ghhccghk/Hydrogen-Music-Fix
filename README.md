## 👷‍♂️ 打包客户端

~~由于个人设备限制，只打包了Windows平台的安装包且并未适配macOs与Linux平台。~~
本分支已经完成了Windows平台和Linux平台的打包，macOS由于个人设备限制暂未打包。
如有可能，您可以在开发环境中自行适配。

```shell
# 打包
npm run dist
```

## :computer: 配置开发环境

运行本项目

```shell
# 安装依赖
npm install

# 运行Vue服务
npm run dev

# 运行Electron客户端
npm start
```
